如何添加课程

使用场景说明

孩子要开始学新内容前，需要先把要学的课加进 App。加好课程后，才能在「卡片」「听写」「运用」里练习词汇。

功能说明

WordHub 按小四、小五、小六整理好了华文词汇课程。你可以从「可选课程」里挑选，加入「我的课程」。加好后，孩子随时可以在不同课程之间切换。

操作步骤

  1. 点底部 「学习」 标签，进入学习页面。
  2. 点页面顶部的 「课程」，打开课程学习界面。
  3. 点 「➕ 课程管理」，在弹窗右侧「可选课程」里，按年级找到要学的课，点 「添加」；已加入的课会显示 「已添加」。
  4. 关闭弹窗后，在课程栏用 「年级」 和 「📚 选择课程」 切换到刚加好的课，就可以开始学习了。

常见问题 / 小提醒

  • 想一次加一整册？在「可选课程」里可以点 「全部添加」。
  • 想删掉某门课？在「我的课程」列表里向左滑动该课程即可删除。
  • 还没加任何课时，部分功能会提示 「请先添加课程」，先完成添加即可。
  • 首页点 「进入」（学生身份）也会直接打开 「学习 · 课程」，方便快速选课。
